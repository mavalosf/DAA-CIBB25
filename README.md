# DAA-CIBB25
Data and R code used in Avalos et al. A Differential Abundance Analysis Framework for the Indoor Microbiome in Asthma. In proceedings of Computational Intelligence Methods for Bioinformatics and Biostatistics, CIBB 2025. LNBI Springer (Submitted). Genoscreen authorized the release of these OTU tables and metadata from the research project Kamyre
